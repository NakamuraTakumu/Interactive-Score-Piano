# Interactive Score Piano

> **Note**
> This application was developed using **Gemini CLI**.

## Overview

A web-based application that displays sheet music in MusicXML (.mxl) format.

### Key Features

1.  **Score Display**: Renders `.mxl` files directly in the browser.
2.  **MIDI Input & Highlighting**: Sounds are triggered by MIDI keyboard input, and the corresponding notes on the score are highlighted in real-time.
